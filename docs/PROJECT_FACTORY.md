# CTK OS Project Factory

Create projects with:

```bash
ctk new website "My Website" ../my-website
ctk new ai-app "My AI App" ../my-ai-app
ctk new music-release "My Release" ../my-release
ctk new education-app "My Education App" ../my-education-app
ctk new game "My Game" ../my-game
```

Every generated project includes `metadata.csv`, `catalog.csv`, `project.yaml`, versioning files, and a basic GitHub Actions validation workflow.

Validate a generated project with:

```bash
ctk check ../my-website
```
