# Changelog

## v0.7.0 — Music Intelligence Foundation

- Added local deterministic WAV analysis and structured audio-analysis CSV output.
- Added local K-Means clustering without a paid AI dependency.
- Added TOAST mini-EP recommendations with evidence, alternatives, confidence, cost, and approval.
- Added Creative DNA catalog summaries.
- Added optional richer music-analysis dependency group.
- Added music architecture documentation and automated tests.
